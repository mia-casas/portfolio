CSS Creature

Within the first few days of the Eleven Fifty Academy Bootcamp, students are equipped with basic manipulation skills to create a two-dimensional smiley face and are encouraged to experiment making their own personal "css creature." For my first Gold Badge Project I choose to create a "modern" interpretation of Walt Disney's beloved Mickey Mouse.

Static Storefront

For the second Gold Badge Project students must troubleshoot making a static storefront, or other webpage, to present in their portfolio. Students practice creating a wireframe design to guide the layout of their webpage that consists of HTML elements and CSS styling. My store front features a fictional online retail page for women. The photos promote the artwork of Nina Tadic Photography and her talent for capturing beautiful, real people.

Google Maps Guided API

As an introduction to API's, students created a webpage that interacts with the Google Maps API to return data to the user. In this tutorial, the API returns data such as a user's location by crating a custom map.

Giphy API

Our final and most challenging project required that we create a webpage that generates content from a third-party API. I choose to create a safe site that allows users to search for gifs by inputting a keyword, which would generate friendly gifs with a G/PG rating for users to enjoy.
